## Task 2 – Digital Portfolio (Reflection)

For this task, I created and updated my profiles on GitHub, LinkedIn, and Kaggle to start building a professional digital presence. GitHub helped me understand how to showcase my work and organize projects, while LinkedIn introduced me to the importance of maintaining a formal academic and career profile. Kaggle was particularly interesting as it focuses on data science and real-world problem solving.

Each platform serves a different purpose — GitHub for project hosting, LinkedIn for networking, and Kaggle for skill development and competitions. Setting up these profiles made me realize how important it is to maintain a consistent and professional online identity.

Over the next four years, I plan to regularly update these platforms by adding projects, participating in challenges, and connecting with like-minded individuals. This will help me track my growth, improve my skills, and prepare for future internship and career opportunities.
